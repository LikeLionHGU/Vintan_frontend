## Service Introduction

### 공실 상권 분석과 현장 분위기를 연결하는 로컬 상권 커뮤니티 플랫폼, '빈땅'

> - 😔 예비 창업자와 업종 전환·확장을 고민하는 기존 자영업자는 상권 분석과 입지 판단에 필요한 정보와 경험이 부족해 점포 선택에 어려움을 겪고, 이는 상가 공실률 증가로 이어져 공간 활용도 저하, 상권 쇠퇴라는 문제를 발생시킨다.

### 💡 이러한 문제를 해결하기 위해,

> 1. AI 분석을 통해 경쟁업체 분포, 유동인구 흐름, 접근성 및 주변 시설 환경 등 다양한 데이터를 종합적으로 검토하여, 공실에 가장 적합한 업종을 객관적으로 제안합니다.

> 2. 상권의 분위기를 데이터로 읽어내고 실질적 경험을 바탕으로 예비창업자의 성공을 지원하는 커뮤니티입니다.


## Stack

### 🌟 Frontend(Web)
> Frontend: React, JavaScript <br/>
> Styling: Material-UI, Emotion <br/>
> State Management: Zustand <br/>
> Routing: React Router <br/>
> Data Fetching: Axios <br/>
> Charts: Chart.js, Recharts <br/>
> Deployment: Firebase Hosting <br/>
